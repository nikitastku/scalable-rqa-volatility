================================================================================
  F. KEY FINDINGS FOR RQ3 (non-stationarity stress test)
================================================================================

1. DEEP LEARNING DOMINATES: LSTM with RV features achieves AUC=0.975,
   far ahead of all other approaches. Adding lagged RV as input was the
   single most impactful improvement (+0.046 F1 for GINN at λ=0.3).

2. TEACHER DISTILLATION FAILS: GINN's GJR-GARCH teacher (AUC=0.888)
   is worse than the student. Distillation hurts monotonically:
   λ=0.0 (0.975) > λ=0.1 (0.974) > λ=0.3 (0.968) > λ=0.5 (0.954).

3. RQA FEATURES AT CHANCE: All RQA-only and β-RQA-only variants perform
   near random (AUC 0.48–0.56). The extreme distribution shift means
   recurrence patterns calibrated on high-volatility training data
   carry no discriminative signal in the calm test period.

4. RQA CONFIG INSENSITIVE: Varying window (30/60/90), embedding (m=3-6,
   tau=1-3), and mode (joint/per_series) made virtually no difference.
   RF Std+RQA hovers at AUC 0.877–0.879 regardless of config.

5. β SELECTION MATTERS SLIGHTLY: β=1.5 is best for RF Std+β-RQA
   (AUC=0.874), but all β values underperform RF Std alone (0.879).
   The horizontal line measures (LAM_h, TT_h) provide modest benefit
   for β-RQA-only (F1 improved vs old 6-feature version).

6. STANDARD FEATURES SUFFICIENT: Simple rolling statistics (ret_abs,
   ret_sq, rv, rolling means/stds) with RF capture nearly all
   predictable signal on this dataset (AUC=0.879).

CONCLUSION FOR DATASET 1: Under extreme non-stationarity, recurrence-
based features add no value. The real test of RQA/β-RQA will be
Dataset 2, where train and test share similar distributional properties.

   at β=5.0 (AUC=0.786), up from 0.742 at β=0.5. Recurrence features are
   complementary to standard features, not standalone replacements.

   503 stocks overwhelms per-window differences.

5. β=2.0 IS BEST FOR STANDALONE β-RQA: LogReg β-RQA-only at β=2.0
   achieves AUC=0.672, the best standalone recurrence result. This
   is Euclidean distance — on intraday data, the symmetric metric
   slightly outperforms asymmetric variants.

   AUC=0.920 on 25K rows is likely underestimating true performance
   vs the 450K-row feature baselines. Not directly comparable.

OVERALL THESIS CONCLUSION:
  - RQ1: RQA/β-RQA features improve classification on real, stationary data
    (D2, D3), with strongest gains on daily data (D2: +0.034 F1 for LogReg).
    ★ LogReg Std+β-RQA at β=5.0 achieves AUC=0.987, surpassing HAR-RV (0.986).
    ★ RF Std+β-RQA at β=4.0 achieves F1=0.938, best among all feature baselines.
    On intraday data (D3), improvements are marginal (+0.001 AUC) but consistent.
  - RQ2: RQA scales to 3M rows / 503 stocks in ~5-10 minutes with step=20.
  - RQ3: Under extreme non-stationarity (D1), recurrence features fail completely.
  - β-RQA: Optimal β depends on data frequency:
    Daily data (D2): β=4-5 (beyond literature) dramatically outperforms β≤2.
    Intraday data (D3): β=2 (Euclidean) is sufficient; higher β provides no gain.
    This suggests large-value emphasis (high β) benefits lower-frequency data
    where volatility regime transitions are more pronounced.
  - Embedding: m=3-4, tau=2 is robust for daily data. m=5 is best standalone
    for intraday data. Combined models are largely insensitive to m/tau/RR on D3.
  - Key insight: The value of parameter tuning depends on dataset complexity.
    On smaller daily datasets (D2), parameter choice matters significantly.
    On large-scale intraday data (D3), cross-sectional diversity dominates.
